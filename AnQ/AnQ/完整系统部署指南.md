# 完整系统部署指南（IoTDA架构）

## 系统架构

```
香橙派 → 华为云IoTDA → SpringBoot后端 → HarmonyOS前端
  ↓           ↓              ↓              ↓
推理+OBS    MQTT接收      HTTP接收       轮询展示
         数据转发      存储+API      证据链UI
```

## 一、后端部署

### 1. 项目结构
```
backend/
├── AlertApplication.java      # 主类
├── Alert.java                 # 实体类
├── IoTDAMessage.java          # IoTDA消息DTO
├── IoTDAController.java       # 接收IoTDA推送
├── application.properties     # 配置文件
└── pom.xml                    # Maven依赖
```

### 2. 编译运行
```bash
mvn clean package
java -jar target/alert-system-1.0.0.jar
```

### 3. 接口说明
- POST /iotda/callback - 接收IoTDA推送（内部接口）
- GET /alert/latest - 前端查询最新告警
